## Descrição do Projeto:
---
* O intuito desse projeto é de documentar meus estudos sobre Java.
* Neste exemplo, estou utilizando recursos de Spring Boot e usando Java 25.
* Utilizei a dependência do Spring WEB para construir a API e consumir as informações do Viacep.
* Para testar, utilizei o Postman, fazendo um Request Mapping com o que foi criado no projeto para retornar as informações conforme parâmetro informado.
